## ğŸ§  How It Works

The chatbot uses a rule-based sentiment analyzer that:
1. Tokenizes your input into words
2. Matches words against positive/negative dictionaries
3. Calculates a sentiment score (-1 to 1)
4. Generates an appropriate empathetic response
5. Tracks sentiment trends across the conversation
